# GIS-Groundwater-Mapping-Project
Explanation of the Logic (For your Report/Thesis)Rainfall (CHIRPS): We extracted daily rainfall data for the last 5 years (2020-2025) and averaged it. Higher rainfall areas in Mymensingh get a rank of 5.Slope (SRTM DEM): Mymensingh is mostly flat, but subtle slope changes matter. We ranked 0-2 degrees (flat) as 5, and steeper areas as lower potential.LULC (ESA WorldCover): We used European Space Agency data. "Cropland" and "Water bodies" allow infiltration (Rank 5), while "Built-up" (Urban areas like Mymensingh Sadar) blocks water (Rank 1).Weighted Overlay: We combined these layers using the formula:$$GWPZ = (Rain \times 0.35) + (Slope \times 0.20) + (LULC \times 0.25) + (Drainage \times 0.20)$$Limitations & Future WorkTo make this "Thesis Quality," you need to add one layer that exists offline:Lithology/Geology: The Google Cloud does not have a detailed geological map of Bangladesh. In your actual project, you would digitize the Madhupur Clay vs. Alluvial Sand map in ArcGIS and upload it as a shapefile to merge with this code.
